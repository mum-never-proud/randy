randy

Random ID generator

The current buffer can hold upto 10,000 entries, post which the buffer is cleared.

At any point the collisions can be detected upto 10,000 numbers.

Collisions (0)

Syntax:

          
            /**
             * @param {string} len [len = 5] - length of the random string
             * @param {string} base [base = 36] - any valid numerical base
             */
            window.randy(len, base);
